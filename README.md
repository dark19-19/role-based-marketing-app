How to Run

1- Set up Postgres and create the `veronicaDB` database (or adjust DATABASE_URL ).


2- Copy
 ```.env.example``` to ```.env``` and update values.


3-make sure that you are in backend directory.


4- Run ```npm install``` .


5- Run ```npm run migrate``` to apply core migrations.


6- Run ```npm run dev``` to start the server.

7- Open Postman  and go to imports

8- import the postman collection.
